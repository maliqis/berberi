## ğŸ’» Usage

### For Customers

1. **Sign Up / Login** - Create an account or log in with existing credentials
2. **Browse Shops** - Explore available barbershops on the home screen
3. **Select Barbershop** - Tap on a barbershop card to view details
4. **Book Appointment**:
   - Choose a date
   - Select a time slot
   - Optionally select a specific barber or use auto-select
   - Add your name and any special requests
   - Confirm booking
5. **Manage Reservations** - View all your appointments in the "My Reservations" tab

### For Barbershop Admins

1. **Sign Up / Login** - Register as a barber shop owner
2. **Configure Shop** - Set up shop name, logo, working hours, and working days
3. **Add Employees** - Add barbers/employees to your shop
4. **Manage Schedule** - View and manage all appointments in the schedule timeline
5. **Settings** - Adjust shop settings, employee details, and subscription plan
